Select下拉选择

Select是对浏览器原生select的美化和增强。原先的select或input将被隐藏，选项的值可以通过原先的元素的value属性取得。


初始化

你必须使用javascript初始化下拉选择。

    
	$('select').select();

通过html生成选项

	<!-- html -->
	<select id="select1">
		<option value="1">苹果</option>
		<option value="2">橘子</option>
		<option value="3" disabled>香蕉</option>
		<option value="4">bb</option>
	</select>
	// javascript
	$('#select1').select();

通过js生成选项

通过js生成选项时，也可以使用<input>元素。


	<!-- html -->
	<input type="text" id="select2">
	// javascript
	$('#select2').select({
		data: [
			{ value: 1, text: 'apple' },
			{ value: 2, text: 'orange' },
			{ value: 3, text: 'banana' },
			{ value: 4, text: 'tomato', disabled: true }
		]
	});

多选

	<!-- html -->
	<select id="select5">
	// javascript
	$('#select5').select({
		multiple: true,
		data: [ ... ]
	});

通过js指定初始值

	<!-- html -->
	<select id="select6">
	<select id="select7">
	<select id="select8">
	// javascript

	// 通过initialValue指定初始值
	$('#select6').select({
		initialValue: 2,
		data: [
			{ value: 1, text: 'apple' },
			{ value: 2, text: 'orange' },
			{ value: 3, text: 'banana' },
			{ value: 4, text: 'tomato' }
		]
	});

	// 通过data指定初始值
	$('#select7').select({
		data: [
			{ value: 1, text: 'apple' },
			{ value: 2, text: 'orange' },
			{ value: 3, text: 'banana', selected: true },
			{ value: 4, text: 'tomato' }
		]
	});

	// 多选，指定初始值
	$('#select8').select({
		multiple: true,
		data: [
			{ value: 1, text: 'apple', selected: true },
			{ value: 2, text: 'orange' },
			{ value: 3, text: 'banana', selected: true },
			{ value: 4, text: 'tomato' }
		]
	});

配置项

名称 类型 默认值 描述
position String '' 下拉列表的方向，'up'为向上
multiple Boolean false 是否多选
data Array/Object [] Select的选项
initialValue String 空 初始值
onChange Function function(value, text) {} 选中项目改变时触发
参数1：value: 选中项目的值
参数2：text: 选中项目的文本

API

禁用 & 启用


   
	$('#select10').select('disable');  // 禁用
	$('#select10').select('enable');  // 启用

通过javascript选择项


   
	$('#select12').select('setValue', 2);

重新设置选项


   
	$('#select11').select('setData', [
		{ value:1, text:'football', selected:true },
		{ value:2, text:'swim' }
	]);
