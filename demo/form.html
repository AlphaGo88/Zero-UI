Form表单

基本实例

.z-form-row代表表单的行。

给文本框等元素添加类名.z-form-control。

将表单元素置于.z-form-control-wrapper内。

给.z-form-label添加类.required以显示必填星号。

<form>标签不是必须的。如果想避免表单自动提交，请将.z-form-action置于<form>之外。


如果不使用栅格布局，你需要通过内联样式或外部css使表单元素对齐。

表单校验

你可以使用 jQuery-Validate 插件进行表单校验。

注意：MUI不包括jQuery-Validate，如果需要使用请单独引入。


    <!-- html -->
    <form id="my-form" style="width: 300px">
        <div class="z-form-row">
            <label class="z-form-label required z-col-3">姓名：</label>
            <div class="z-form-control-wrapper z-col-9" style="width: 280px">
                <input type="text" name="name" class="z-input z-form-control">
            </div>
        </div>
        <div class="z-form-row">
            <label class="z-form-label required z-col-3">年龄：</label>
            <div class="z-form-control-wrapper z-col-9" style="width: 280px">
                <input type="text" name="age" class="z-input z-form-control">
            </div>
        </div>
        <div class="z-form-action z-col-9 z-offset-3">
            <button class="z-btn primary medium" id="submit">提交</button>
        </div>
    </form>
    // javascript
    var validator = $('#my-form').validate({
        rules: {
            name: 'required',
            age: 'required'
        },
        messages: {
            name: {
                required: '请输入姓名'
            },
            age: {
                required: '请输入年龄'
            }
        }
    });

    $('#submit').on('click', function(e) {
        validator.form();
    });
