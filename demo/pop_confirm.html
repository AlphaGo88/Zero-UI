PopConfirm弹出确认

在执行某项操作时使用，常用于删除操作。


基本实例


	$('#my-btn').on('click', function(e) {
	    $(this).popConfirm({
	    	content: '确认删除这条记录吗？',
	    	onOK: function() {
	    		// Your code goes here...
	    		Z.Message.success('删除成功！');
	    	}
	    })
	});

配置项

名称 类型 默认值 描述
position String 'bottom' 弹出框的显示方位
有效值：'top'，'bottom'，'left'，'right'
content String 空 弹出框的文字
onOK Function function() {} 点击确认后执行
