Open Dialog

        }

Customize buttons

Open Dialog

你可以自定义按钮，按如下格式把它们添加到actions数组中。

Dialog对象会被传给自定义按钮的onClick函数。


    Z.dialog({
        title: '标题',
        content: content,
        actions: [{
            tmplt: '<button class="z-btn primary">按钮1</button>',
            onClick: function(dlg) {
                alert('你点击了按钮1！')
            }
        }, {
            tmplt: '<button class="z-btn default">按钮2</button>',
            onClick: function(dlg) {
                alert('你点击了按钮2！')
            }
        }, {
            tmplt: '<button class="z-btn red">关闭</button>',
            onClick: function(dlg) {
                dlg.close()
            }
        }]
    });

Content is HTML

Open Dialog

通过html生成对话框的内容。

如果对话框中包含其他组件，请在onOpen()函数中初始化，通过dlg.el访问对话框的jQuery对象。

    <!-- html -->
    <div id="form" style="display: none">
        <form class="form" style="margin-left: -35px">
            <div class="form-row">
                <label class="form-label col-2">姓名：</label>
                <div class="form-control-wrapper col-4">
                    <input type="text" name="name" class="form-control">
                </div>
                <label class="form-label col-2">邮箱：</label>
                <div class="form-control-wrapper col-4">
                    <input type="text" name="email" class="form-control">
                </div>
            </div>
            <div class="form-row">
                <label class="form-label col-2">生日：</label>
                <div class="form-control-wrapper col-4">
                    <input type="text" name="birth" class="form-control">
                </div>
                <label class="form-label col-2">住址：</label>
                <div class="form-control-wrapper col-4">
                    <input type="text" name="addr" class="form-control">
                </div>
            </div>
          </div>
        </form>
    </div>
    // javascript
    Z.dialog({
        title: '标题',
        width: 600,
        content: $('#form').html(),
        onOpen: function(dlg) {
            dlg.el.find('[name=birth]').datetimepicker();
        }
    });

Events

回调函数的第一个参数是实例化的Dialog对象。


Open Dialog 

   Z.dialog({
       title: '标题',
       content: content,
       onOpen: function(dlg) {
           alert('对话框将显示！')
       },
       onOK: function(dlg) {
           alert('你点击了确认按钮！')
       }
   });

配置项

名称 类型 默认值 描述
title String 空 对话框的标题
width Number/String 页面宽度的60% 对话框的宽度。
例如：500 或 '500px'。
content Html/String 空 对话框的内容
okButton Boolean true 是否显示“确认”按钮
cancelButton Boolean true 是否显示“取消”按钮
actions Array [] 自定义对话框的按钮。
如何使用参见实例。
onOpen Function function(dlg) {} 对话框打开时触发
参数：dlg: Dialog对象
onClose Function function(dlg) {} 对话框关闭时触发
参数：dlg: Dialog对象
onOK Function function(dlg) {} 点击“确认”按钮时触发
参数：dlg: Dialog对象

API

名称 返回值 描述
dlg.close() 关闭对话框，并销毁。
